# Серверное приложение в роли прокси-сервера мессенджера
Для запуска данного проекта необходимо:
1. Скачать проект полностью
2. Установить всё командой npm install
3. Добавить файл .env и написать свои следующие параметры по такому шаблону: 
DB_HOST= 
DB_PORT=
DB_NAME=
DB_USER=
DB_PASS=
PORT=
4. Запускаете серверное приложение при помощи команды npm run start
